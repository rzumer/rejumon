# Reじゅもん

ドラゴンクエスト　ふっかつのじゅもん　修正ツール

このツールは、単なるパスワード生成ツールではなく、自動的に誤字を検出して修正するツールです。
紙に書いたパスワードが誤っていた場合などに役立つように作りました。

※誤字は１つまで検出する<br>
※DQ2の形式はまだ実装されていない
